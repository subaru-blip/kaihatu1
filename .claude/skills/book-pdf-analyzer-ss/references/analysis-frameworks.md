# ジャンル別分析フレームワーク

## 1. ビジネス・マーケティング書

### 検出すべきフレームワーク

| カテゴリ | フレームワーク | キーワード |
|----------|--------------|-----------|
| 戦略 | SWOT分析 | 強み、弱み、機会、脅威 |
| 戦略 | ポーターの5フォース | 競争環境、参入障壁、代替品 |
| 戦略 | ブルーオーシャン戦略 | 価値曲線、差別化、新市場 |
| マーケティング | 4P/7P | Product, Price, Place, Promotion |
| マーケティング | STP | セグメンテーション、ターゲティング、ポジショニング |
| 購買行動 | AIDMA/AISAS | 注意、興味、欲求、記憶、行動 |
| 購買行動 | カスタマージャーニー | 認知、検討、購入、利用、推奨 |
| 経営 | ビジネスモデルキャンバス | 価値提案、顧客セグメント、収益 |
| 経営 | OKR/KPI | 目標、成果指標、重要業績 |
| 集客 | ファネル | TOFU, MOFU, BOFU、認知→獲得→転換 |
| ブランド | ポジショニングマップ | 軸、差別化、独自性 |
| 成長 | グロースハッキング | AARRR、バイラル係数 |

### 分析観点

1. **著者の主要メソッド**: 著者独自の手法・理論は何か
2. **ターゲット顧客像**: 誰に向けた戦略か
3. **数字・実績**: 具体的な成果データ
4. **再現性**: 読者が実行できるか
5. **競合との差別化**: 他の書籍・手法との違い

## 2. 技術書・ハウツー本

### 分析すべき要素

| 要素 | 説明 | 例 |
|------|------|-----|
| 前提知識 | 読者に求められるスキルレベル | 初心者/中級者/上級者 |
| ツール一覧 | 必要なソフトウェア・道具 | Photoshop, CLIP STUDIO等 |
| 手順フロー | 全体の制作/実装フロー | 企画→下書き→ペン入れ→仕上げ |
| テクニック集 | 個別のテクニック・Tips | パース技法、色彩理論等 |
| 作例・サンプル | 実例の数と種類 | 完成作品、途中経過 |
| 練習課題 | 読者向けの演習 | チャレンジ問題、ワーク |

### テクニック分類テンプレート

```
| No. | テクニック名 | 概要 | 難易度 | 該当ページ |
|-----|-------------|------|--------|-----------|
| 1   | {name}      | {desc} | 初級  | p.XX      |
```

### 手順抽出テンプレート

```
## 全体フロー（書籍通し）

### Step 1: {大工程名} (第X章)
- {具体的な手順}
- {ポイント・注意点}

### Step 2: {大工程名} (第X章)
...
```

## 3. 自己啓発・教育書

### 分析すべき要素

| 要素 | 説明 |
|------|------|
| 中心メッセージ | 著者が最も伝えたい1つのこと |
| マインドセット変化 | Before→Afterの意識変革 |
| 習慣化の仕組み | 具体的な行動変容の方法 |
| エビデンス | 科学的根拠・研究データ |
| ストーリー/事例 | 成功事例・失敗事例 |
| ワーク/実践 | 読者が取り組むべき演習 |

### 行動変容テンプレート

```
## 行動変容マップ

### Before（読む前の状態）
- {typical mindset/behavior}

### After（読んだ後の理想状態）
- {target mindset/behavior}

### 変化のステップ
1. {awareness} - 気づき
2. {understanding} - 理解
3. {practice} - 実践
4. {habit} - 習慣化
```

## 4. 汎用（ジャンル不明時）

### 基本分析要素

| 要素 | 説明 |
|------|------|
| テーマ | この本の主題は何か |
| 著者の立場 | 著者はどの視点から書いているか |
| 論の構造 | 問題提起→分析→解決策→結論 |
| 対象読者 | 誰に向けて書かれているか |
| 独自の貢献 | この本ならではの知見は何か |
| 限界・注意点 | 書かれていないこと、バイアス |

### 汎用分析テンプレート

```
## 総合分析

### 著者の主要メッセージ
{本全体を通じて最も伝えたいこと}

### 論理構造
{問題提起 → 根拠 → 主張 → 結論}

### 主要な根拠・エビデンス
1. {evidence 1}
2. {evidence 2}

### 対象読者と前提知識
{この本が最も役立つ人}

### 独自の貢献
{他の書籍にない知見}
```
