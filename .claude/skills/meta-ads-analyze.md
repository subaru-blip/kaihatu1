# Meta Ads Analyze Skill

Meta広告のパフォーマンスを深く分析し、改善提案を生成するスキル。

## 概要

このスキルは以下の分析を実行します：

1. **パフォーマンスサマリー** - 主要KPIの概要
2. **トレンド分析** - 時系列での推移
3. **セグメント分析** - 年齢/性別/配置別
4. **異常検出** - パフォーマンス急変の検出
5. **改善提案** - データに基づく具体的アクション

## 使用方法

```
/meta-ads-analyze [期間] [オプション]
```

### 例

```
# 過去7日間の分析
/meta-ads-analyze 7d

# 過去30日間の詳細分析
/meta-ads-analyze 30d --detailed

# 特定キャンペーンの分析
/meta-ads-analyze --campaign-id 123456789

# 競合比較付き分析
/meta-ads-analyze 30d --with-competitors
```

## 分析項目

### 1. 主要KPI

| 指標 | 説明 | 目標基準 |
|------|------|----------|
| ROAS | 広告費用対効果 | 3.0以上 |
| CPA | 獲得単価 | 業界平均以下 |
| CTR | クリック率 | 1.0%以上 |
| CVR | コンバージョン率 | 2.0%以上 |
| Frequency | フリークエンシー | 3.0以下 |

### 2. セグメント分析

```
年齢別パフォーマンス:
├── 18-24: CTR 1.2%, CPA ¥2,500
├── 25-34: CTR 1.8%, CPA ¥1,800 ★Best
├── 35-44: CTR 1.5%, CPA ¥2,100
└── 45+:   CTR 0.9%, CPA ¥3,200

配置別パフォーマンス:
├── Feed:    50% spend, ROAS 3.2 ★Best
├── Stories: 30% spend, ROAS 2.8
├── Reels:   15% spend, ROAS 2.1
└── Other:   5% spend,  ROAS 1.5
```

### 3. 異常検出

- CTR急落（前週比-20%以上）
- CPAスパイク（前週比+30%以上）
- フリークエンシー過多（>5.0）
- クリエイティブ疲労（CTR継続低下）

## 内部処理フロー

```
1. meta-ads MCP → Insights API呼び出し
2. データ集計・正規化
3. 統計分析（トレンド、異常検出）
4. data-analyst エージェント → 詳細分析
5. 改善提案生成
6. レポート出力
```
