# manga-produce-creator-ss

電子書籍を漫画化することに特化したスキル。

## 特徴

- **章ベースで漫画化**: 書籍の各章を独立した漫画シーンとして構成
- **各章3-5ページ**: 全5章なら15-25ページの漫画が完成
- **重要ポイント抽出**: 章の主旨が伝わる部分を優先、細部は省略OK
- **対話形式**: 主人公（学ぶ役）と先生（解説役）の会話で展開

## ebook-creator-ss との連携

```
1. ebook-creator-ss で電子書籍作成（15,000字、5章構成）
2. manga-produce-creator-ss で書籍を漫画化（15-25ページ）
```

完成物:
- 電子書籍（DOCX + 図解）
- 漫画版（各章3-5ページ）

→ 同じ内容を文章と漫画の両方で提供可能

## 使用方法

```
「書籍を作った後に漫画作りたい」
「この電子書籍を漫画化して、各章3-5ページで」
```

詳細は `SKILL.md` を参照。
