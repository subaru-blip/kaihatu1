# Kindle出版スキル

## When to Use
- Kindle本の執筆
- 電子書籍の構成設計
- 目次・章立て作成

## 参照コンテンツ
- taiyou-taiyo/content/kindle/ (10章構成テンプレート)

## 構成
1. 概要（アウトライン）

3. 第1章〜第7章
