## Context
- Repo root: !`git rev-parse --show-toplevel 2>/dev/null || pwd`
- Current branch: !`git branch --show-current 2>/dev/null || echo "(no-git-branch)"`
- Working dir: !`pwd`

## Goal
Decide:
1) spec-slug
2) target-dir
then print the exact next command to run: `/sdd-req100 <spec-slug> <target-dir>`

## Rules (deterministic)
- If `$1` is provided, set `spec-slug = $1`.
- Else, derive `spec-slug` from current branch name (preferred) or repo dir name:
  - kebab-case
  - remove prefixes: feature/, feat/, bugfix/, hotfix/, chore/
  - replace `/` with `-`
- If `$2` is provided, set `target-dir = $2`.
- Else, set `target-dir = .kiro/specs/<spec-slug>/`

## Output format (STRICT)
Print exactly 3 lines and nothing else:
```
spec-slug: <value>
target-dir: <value>
next: /sdd-req100 <spec-slug> <target-dir>
```
