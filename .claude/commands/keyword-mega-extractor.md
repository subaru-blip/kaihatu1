# /keyword-mega-extractor

リサーチ結果からキーワードを自動抽出。ロングテール・ニッチ・売れるキーワードを発見。

## 使い方

```
/keyword-mega-extractor <シードキーワード> [--type=all|longtail|niche|trending|buying]
```

## 例

```
/keyword-mega-extractor 投資信託
/keyword-mega-extractor プログラミング --type=longtail
/keyword-mega-extractor 転職 --type=buying
```

## キーワードタイプ

| タイプ | 説明 | 用途 |
|--------|------|------|
| all | 全種類 | 総合分析 |
| longtail | 3語以上 | SEO |
| niche | 低競合 | 穴場狙い |
| trending | 急上昇 | トレンド |
| buying | 購買意図高 | アフィリエイト |
