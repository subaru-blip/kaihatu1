## 7-1. メールを自動で仕分け・返信案を作成

**こんな人におすすめ：** 毎日大量のメールが来て処理しきれない方

OpenClawにメール管理を任せると、以下を自動で行ってくれます。

![メール自動仕分けの流れ](images/ch07_01_email_sorting.png){ width=100% }

1. メールを全件チェックします
2. 緊急度を判断します（取引先の急ぎの連絡、営業メール、スパムなど）
3. 緊急なものから順にリストアップします
4. それぞれに返信の下書きを作成します
5. 「確認してください」と報告します

あるユーザーは4,000通以上たまったメールを2日間でOpenClawに処理させました。スパムの解除、緊急度別の分類、返信案の作成を夜間に自動実行したという事例があります。

## 7-2. 毎朝、今日の予定と天気をお知らせ

**こんな人におすすめ：** 朝バタバタして予定を確認する余裕がない方

![朝のブリーフィング内容](images/ch07_02_briefing_tree.png){ width=100% }

設定しておくと、毎朝自動でこのようなメッセージが届きます。

> OpenClaw: 「おはようございます。今日のブリーフィングです。【天気】晴れ→午後から曇り（最高18℃）【予定】10:00 営業ミーティング / 15:00 クライアント来社 【リマインド】明日までに企画書の修正が必要です」

頼まなくても自動で連絡してくれます。これがOpenClawの真骨頂です。

## 7-3. レシートの写真を撮るだけで家計簿に記録

**こんな人におすすめ：** 家計簿をつけたいけど面倒で続かない方

レシートの写真を撮ってチャットで送るだけで、以下を自動で行ってくれます。

![レシート家計簿の流れ](images/ch07_03_receipt_flow.png){ width=100% }

1. 写真からお店の名前、日付、金額を読み取ります
2. カテゴリを判断します（食費、日用品、交通費など）
3. スプレッドシートに自動で記録します
4. 月末に集計レポートを作成します

レシートの写真を送ると「記録しました。セブンイレブン、食費、638円。今月の食費合計は32,450円です」と返ってきます。写真を送るだけなので、誰でも続けられます。

## 7-4. スマホからメッセージを送るだけでWebサイトを更新

**こんな人におすすめ：** ブログやWebサイトを持っているが更新が面倒な方

![Webサイト更新の流れ](images/ch07_04_web_update_flow.png){ width=100% }

チャットでテーマと大まかな方向性を伝えるだけで、記事の作成からブログへの投稿まで自動で行ってくれます。あるユーザーは、ベッドでくつろぎながらスマホからOpenClawにメッセージを送るだけでWebサイトのリニューアルを完了させました。

## 7-5. 会議の録画を自動で要約

**こんな人におすすめ：** 会議に参加できなかった時に録画を見直す時間がない方

![会議要約の出力形式](images/ch07_05_meeting_summary.png){ width=100% }

OpenClawに依頼すると、以下の形式で要約してくれます。

- **議題**：何について話し合ったか
- **決定事項**：何が決まったか
- **アクションアイテム**：誰が何をいつまでにやるか
- **次回の会議予定**

1時間の会議を3分で把握できます。

## 7-6. 音声で話しかけるだけで操作できる

**こんな人におすすめ：** タイピングが苦手な方、手が離せない時にも使いたい方

![音声操作の方法](images/ch07_06_voice_control.png){ width=100% }

OpenClawは音声操作にも対応しています。

- WhatsAppの音声メモで指示を送ると、文字に変換して処理し、音声で返答してくれます
- Androidでは音声アシスタント化するアプリ（OpenClawAssistant）もあります
- 「Hey OpenClaw」のようなウェイクワード設定も可能です

料理中に「明日の買い物リスト作って」と声をかけると、「買い物リストを作成しました。卵もそろそろなくなりそうですが、追加しますか？」と返してくれます。手を使わなくていいのが便利です。

## 活用のコツ

![活用のコツ](images/ch07_07_tips_comparison.png){ width=100% }

| やるべきこと | やってはいけないこと |
|---|---|
| 簡単なタスクから試してください | いきなり銀行操作などを任せないでください |
| うまくいったら複雑なタスクへ進んでください | 最初から完璧を求めないでください |
| 「毎日やること」を自動化してください | AIが間違わないと思い込まないでください |
| 大事な操作は「確認してから実行」設定にしてください | 承認なしで何でもやらせないでください |
