## Q1. プログラミングの知識は必要ですか？

OpenClawの「利用」には不要です。「インストール」にはほんの少し必要ですが、本書第6章の手順通りにコマンドをコピー&ペーストするだけで完了します。

![必要な知識レベル](images/ch10_01_knowledge_levels.png){ width=100% }

| 場面 | 必要な知識 |
|---|---|
| OpenClawを**使う** | 不要です。チャットで日本語を送るだけです |
| OpenClawを**インストール** | コマンドのコピー&ペーストができればOKです |
| OpenClawを**カスタマイズ** | 少し必要です（設定ファイルの編集など） |

## Q2. 料金はどれくらいかかりますか？

OpenClaw本体は無料です。AIの利用料（API）が使った分だけかかります。

![料金の内訳](images/ch10_02_pricing.png){ width=100% }

| 項目 | 料金 |
|---|---|
| OpenClaw本体 | 無料です（オープンソース） |
| AI利用料（API） | 月1,000〜5,000円です（使い方によります） |
| VPS（任意） | 月500〜1,000円です（24時間使いたい場合のみ） |

### AIごとの料金比較

![AIごとの料金比較](images/ch10_03_ai_pricing.png){ width=100% }

| AI | 月額の目安 | 特徴 |
|---|---|---|
| Claude API | 1,000〜5,000円 | OpenClawとの相性が最も良いです |
| ChatGPT API | 1,000〜5,000円 | 万能型です |
| DeepSeek API | 300〜1,000円 | コスパ最強です |
| ローカルLLM（Ollama） | **完全無料** | 性能は落ちますが料金ゼロです |

**節約テクニック：** 月額上限を必ず設定してください。DeepSeekやローカルLLMでコストを下げられます。

## Q3. スマホだけで使えますか？

スマホだけでの利用は難しいですが、スマホから「操作」はできます。

![セットアップと日常利用](images/ch10_04_smartphone.png){ width=100% }

| 場面 | 必要なもの |
|---|---|
| セットアップ | パソコンが必要です |
| 日常の利用 | スマホからチャットで指示を送ればOKです |

最初のインストールはパソコンで行います。一度セットアップすれば、日常はスマホのTelegramやDiscordから操作できます。VPSで動かしていれば、パソコンの電源を切っても使い続けられます。

## Q4. 日本語で使えますか？

**完全に日本語対応**しています。AIの頭脳（Claude、ChatGPTなど）が日本語を理解するため、日本語で指示を送り、日本語で返答を受けられます。日本語のファイルも処理できます。

設定画面やコマンドは英語ですが、本書の手順通りにコピー&ペーストすれば問題ありません。

## Q5. 安全に使うための最低限のルールは？

第4章で解説した5箇条を守ってください。

![安全ルール5箇条（再掲）](images/ch10_05_safety_rules.png){ width=100% }

| # | ルール |
|---|---|
| ① | メインのパソコンでは本格運用しないでください |
| ② | APIキーは絶対に人に見せないでください |
| ③ | 知らない人からのメッセージは処理させないでください |
| ④ | 大事な操作は「承認制」にしてください |
| ⑤ | `openclaw doctor` を定期的に実行してください |

さらに安全にしたい場合は、コマンドの許可リスト設定、MCP（外部連携）の最小化、定期的なアップデートを行ってください。

## Q6. OpenClawとClaude Codeの違いは？

役割が違います。詳しくは第9章で解説しています。

![OpenClaw vs Claude Code（再掲）](images/ch10_06_comparison.png){ width=100% }

| | OpenClaw | Claude Code |
|---|---|---|
| 役割 | AI秘書です | AIプログラマーです |
| できること | メール・予定・情報収集 | プログラムを書きます |
| 使い方 | チャットで指示します | ターミナルで指示します |
| 料金 | API従量課金です | 月3,000円〜 or API従量です |

**どっちから始めるべきか：** 仕事の効率化 → OpenClaw、アプリやWebサイトを作りたい → Claude Codeです。

## Q7. LINEで使えますか？

**使えます。** ただし設定がやや複雑です。

![LINE接続の手順](images/ch10_07_line_steps.png){ width=100% }

1. LINE Developersでチャネルを作成してください
2. Messaging APIを有効化してください
3. Webhook URLを設定してください（ngrokまたは固定IPが必要です）
4. OpenClawの設定ファイルでLINEチャネル情報を入力してください

初心者はまずTelegramで始めて、慣れてからLINEに挑戦するのがおすすめです。Telegramの方が設定が圧倒的に簡単なためです。

## Q8. MCP（エムシーピー）って何ですか？

**MCP = Model Context Protocol**。AIと外部ツールをつなぐ「共通規格」です。

![MCP登場前と後](images/ch10_08_mcp.png){ width=100% }

USB-Cをイメージすると分かりやすくなります。USB-Cはどんなスマホでも同じケーブルで充電できる共通規格です。MCPはどんなAIでも同じ方法で外部ツールと接続できる共通規格です。

MCPが登場する前は、AIとGoogleカレンダーをつなぐ専用プログラム、AIとSlackをつなぐ別の専用プログラム…とバラバラでした。MCPにより、1つの共通規格で何でもつなげるようになりました。

OpenClawがGoogleカレンダー、Gmail、Notion、Slackなど様々なサービスと簡単に連携できるのは、MCPのおかげです。

## Q9. スキル（Skills）って何ですか？

OpenClawに新しい能力を追加する「プラグイン」のことです。スマホのApp Storeのように、**ClawHub（クローハブ）**というマーケットプレイスで700以上のスキルが配布されています。

![ClawHubのスキル例](images/ch10_09_skills.png){ width=100% }

| スキル例 | 内容 |
|---|---|
| メール管理 | メールを自動分類・返信します |
| カレンダー | 予定管理・リマインドします |
| Web検索 | 最新情報を調査します |
| データ分析 | スプレッドシートの分析をします |
| 買い物 | 価格比較をします |

**注意：** 第4章で触れた通り、悪意あるスキルも報告されています。公式スキルや評価の高いもののみを使い、不明なスキルはインストールしないでください。
