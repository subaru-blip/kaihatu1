# 第2章：OpenClawの3つのスゴさ

\newpage

## ～ 今までのAIにはできなかったこと ～

![OpenClawの3つのスゴさ](images/ch02_header_3features.png){ width=100% }

---

OpenClawがすごいと言われる理由は、大きく分けて3つあります。この3つを知ると、今までのAIが物足りなく感じるかもしれません。

\newpage

## 2-1.「全部覚えてる」～ 昨日の会話も2週間前の話も忘れない ～

Siriに「昨日何を話したか覚えてる？」と聞いても、何も覚えていません。ChatGPTも新しい会話を始めると前回の内容は消えてしまいます。

OpenClawは違います。あなたとの会話を**すべて記憶**しています。先週ポロッと言った「来月引っ越すんだよね」という一言も覚えています。使えば使うほど、仕事のやり方、好み、スケジュールを理解していきます。

![記憶の違い](images/ch02_01_memory_before_after.png){ width=100% }

たとえば「先週話した企画書の件、続きをお願い」と送った場合、ChatGPTは「先週の会話を確認できません」と返しますが、OpenClawは「先週の打ち合わせで決まった方針に基づいて更新しますね。前回の『グラフをもっと入れて』というフィードバックも反映します」と返してくれます。

### なぜ覚えていられるのか

OpenClawはあなたのパソコン（またはサーバー）の中に**会話ログとメモ**を保存しています。クラウド上ではなく手元にデータがあるため、いつでも読み返して思い出すことができる仕組みです。

![OpenClawの記憶の仕組み](images/ch02_02_memory_flow.png){ width=100% }

## 2-2.「向こうから連絡してくる」～ 待つAIから動くAIへ ～

今までのAIは、アプリを開いて話しかけるまでじっと待っているだけでした。

OpenClawは**自分から連絡してきます**。

> OpenClaw: 「緊急のメールが届いています。〇〇社から、明日の会議の時間変更の連絡です」

> OpenClaw: 「先月頼まれたレポートの締め切りが明後日です。まだ未完成のようですが、手伝いましょうか？」

![待つAIと動くAIの違い](images/ch02_03_proactive_before_after.png){ width=100% }

頼まれていなくても、状況に応じて**自分で判断して行動する**のがOpenClawの大きな特徴です。

毎朝こんなメッセージが届くように設定することもできます。

> OpenClaw: 「おはようございます。今日の予定は3件です。10:00 〇〇社ミーティング（資料は昨晩準備済み）/ 14:00 チームランチ（予約確認済み）/ 16:00 週次レポート提出（下書きを作っておきました）。天気は午後から雨です。傘をお忘れなく」

## 2-3.「パソコンを実際に操作できる」～ メール送信もファイル整理もおまかせ ～

これがOpenClawの**最大の特徴**です。

ChatGPTに「メールを送って」と頼んでも、文章を作ってくれるだけで実際にメールを送ることはできません。コピー&ペーストして送るのは自分自身です。

OpenClawは**実際にパソコンを操作して**メールを送ります。ファイルを整理します。ブラウザを開いて情報を調べます。

![OpenClawにできること](images/ch02_04_capabilities_honeycomb.png){ width=100% }

**OpenClawができることの例：**
- メールの送受信、返信
- ファイルの作成・編集・整理
- ブラウザを開いてWebサイトを操作
- スプレッドシートの作成・編集
- カレンダーの予定管理
- チャットアプリでのメッセージ送受信

「ブログの新しい記事を書いて、お問い合わせフォームのメール3通に返信しといて」と送れば、OpenClawがすべて処理してくれます。その間、あなたは好きなことをしていられます。

## 2-4. つまり「AI秘書」が24時間働いてくれる世界

3つの特徴をまとめると次のようになります。

![OpenClawの価値](images/ch02_05_value_formula.png){ width=100% }

| 特徴 | 意味 |
|---|---|
| ① 全部覚えている | あなたのことを理解した上で動いてくれます |
| ② 向こうから連絡 | 頼まなくても気づいて教えてくれます |
| ③ PCを操作できる | 指示するだけで実際に仕事をしてくれます |

この3つが合わさることで、OpenClawは「ツール」ではなく**「一緒に働く相棒」**になります。

今までのAIは「優秀な辞書」でした。聞けば答えてくれますが、自分では何もしません。OpenClawは「優秀な秘書」です。あなたのことを理解して、自分で判断して、実際に手を動かしてくれます。
