## OpenClawを一言でいうと

**「24時間、あなたの代わりに働いてくれるAI秘書」**です。

メールの返信、スケジュール管理、資料作成、データ整理──今まで自分でやっていた面倒な作業を、AIが自動でやってくれる。しかもSiriやChatGPTとは違い、**こちらから話しかけなくても、自分から動いてくれる**のです。

2025年末に登場して以来、世界中で大きな話題になっています。GitHubでは17万スターを超え、関連するMac miniが品切れになるほどの人気ぶりです。

「それ、本当に自分みたいな普通の人でも使えるの？」

はい、使えます。この本を読み終わるころには、きっとそう実感していただけるはずです。

## この本の読み方

この本は全10章＋おわりに、で構成されています。

| 部分 | 章 | 内容 |
|------|-----|------|
| 知る | 第1章〜第2章 | OpenClawとは何か、何がすごいのか |
| 理解する | 第3章〜第4章 | しくみとセキュリティ（安全に使うために） |
| 始める | 第5章〜第6章 | 準備とインストール（手順どおりでOK） |
| 使う | 第7章 | 活用事例集（マネするだけ） |
| 広げる | 第8章〜第9章 | AI SNS「Moltbook」とClaude Code |
| 確認する | 第10章 | よくある質問FAQ |

各章の冒頭には**漫画**がついています。主人公のハルカと一緒に、まずは漫画でストーリーを楽しんでから、詳しい解説を読むとスムーズに理解できます。
