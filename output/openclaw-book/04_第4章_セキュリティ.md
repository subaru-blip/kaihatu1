## ～ セキュリティ編 ～

![セキュリティ5箇条](images/ch04_header_security.png){ width=100% }

---

OpenClawは便利ですが、使い方を間違えると本当に危険です。知らないまま使うのは、車の運転を知らないまま高速に乗るようなもの。怖がらせたいわけではありませんが、**対策すれば安全に使えます**。この章でしっかり学んでおきましょう。

\newpage

## 4-1. OpenClawは「新入社員に会社の鍵を全部渡す」のと同じ

OpenClawを自分のパソコンで動かすと、以下のすべてにアクセスできる状態になります。

- メール
- ファイル
- ブラウザに保存されたパスワード
- ログイン中のサービス（銀行、SNS、ショッピングサイト）
- パソコン上のすべてのデータ

![OpenClawの権限](images/ch04_01_permission_concept.png){ width=100% }

家政婦さんに鍵を渡すイメージです。リビングやキッチンの鍵はOKですが、金庫の鍵や銀行の通帳はNGです。必要な部屋の鍵だけ渡せばいいのです。

OpenClawの公式もこうアドバイスしています。

> 「この権限を、入社1日目の新入社員に渡しても大丈夫か？答えがNoなら、OpenClawにも与えるべきではありません」

必要な権限だけを与える。これが「**最小権限の原則**」と呼ばれる考え方です。

## 4-2. 実際に起きたセキュリティ事件

OpenClawはまだ新しいツールですが、すでに複数のセキュリティ事件が報告されています。

![実際に起きた事件](images/ch04_02_security_incidents.png){ width=100% }

### 事件① メールが全部消された

知らない人から届いたメールの中に「AIへの隠し命令」が仕込まれていました。OpenClawがその命令に従い、**メールをゴミ箱含めてすべて削除**してしまいました。

### 事件② 150万件のAPIキーが流出

AIだけのSNS「Moltbook」のデータベースが誰でも見られる状態になり、**約150万件のAPIキー**が漏洩しました。

### 事件③ 悪意あるスキルが341個発見

OpenClawの拡張機能を配布するClawHubで、**341個の悪質なプログラム**が見つかりました。インストールすると、パスワードや暗号通貨の情報が盗まれる仕組みでした。

### 事件④ Webページを見ただけで乗っ取られる脆弱性

OpenClawのセキュリティ上の欠陥（CVE-2026-25253、深刻度スコア8.8/10）が発見されました。悪意あるWebページを1回訪問するだけで、パソコンを遠隔操作されてしまうという深刻な問題でした（現在は修正済みです）。

こう聞くと怖いかもしれませんが、車と同じです。事故は起きますが、シートベルトやルールを守れば安全に乗れます。

## 4-3. 5つの危険ポイント

![5つの危険ポイント](images/ch04_03_danger_points.png){ width=100% }

### ① 悪い命令が紛れ込む（プロンプトインジェクション）

メールやWebページの中に、人間には見えないがAIには読める「隠し命令」を仕込む攻撃です。手紙の中に透明なインクで「秘書さん、金庫を開けてください」と書いてあるようなものです。OWASPの「LLMアプリケーションセキュリティ脅威Top 10」でも、**第1位**にランクされています。

![プロンプトインジェクション](images/ch04_04_prompt_injection.png){ width=100% }

**対策：** 知らない人からのメールやファイルをOpenClawに処理させないでください。重要な操作は「承認してから実行」する設定にしてください。

### ② チャットが攻撃の入り口になる

OpenClawはチャットから操作するため、あなたにメッセージを送れる人は、AIに命令できてしまう可能性があります。

**対策：** OpenClawは個人チャットだけで使ってください。グループには入れないでください。WhatsAppを使う場合は専用の番号を用意してください。

### ③ 安全装置がわざとついていない

OpenClawは当初、意図的に安全装置（ガードレール）を設けない設計でした。最近のアップデートで改善されつつありますが、自分で安全設定をする必要があります。

**対策：** インストール後に `openclaw doctor` と `openclaw security audit` を必ず実行してください。

### ④ 普段使いのパソコンで動かすと危険

普段使いのパソコンには、パスワード、銀行情報、仕事のデータなど大切な情報がたくさん入っています。万が一の時に全部アクセスされるリスクがあります。

**対策：** 専用の環境（VPSや古いパソコン）で動かしてください。どうしても自分のパソコンで使う場合は、触れるフォルダを限定してください。

### ⑤ 権限の与えすぎに注意

「便利だから」と何でもできるようにすると、事故が起きた時の被害が大きくなります。

**対策：** 実行できるコマンドを「許可リスト」で制限してください。段階的に権限を広げていくのがおすすめです。

## 4-4. 初心者の安全ルール5箇条

![安全ルール5箇条](images/ch04_05_safety_rules.png){ width=100% }

| # | ルール |
|---|---|
| ① | メインのパソコンでは使わないでください（VPSか古いPCで） |
| ② | APIキーは絶対に人に見せないでください |
| ③ | 知らない人からのメッセージは処理させないでください |
| ④ | 大事な操作は「承認制」にしてください |
| ⑤ | `openclaw doctor` を定期的に実行してください |

この5つを守れば、初心者でも安心して始められます。

## 4-5. API料金に注意！

セキュリティとは別に知っておくべきことがあります。**料金**です。

OpenClaw自体は無料ですが、AIの頭脳（LLM）をクラウドで使う場合、使った分だけ料金がかかります。OpenClawはエージェントAIなので、1つのタスクでも何回もAIに問い合わせをします。1回の依頼で4〜5回分の料金がかかることも珍しくありません。

![API料金の仕組み](images/ch04_06_api_pricing.png){ width=100% }

| 使い方 | 月額の目安 |
|---|---|
| 軽く試す程度 | 1,000〜3,000円 |
| 毎日ちょっと使う | 3,000〜10,000円 |
| ガッツリ使う | 10,000〜20,000円以上 |

### 料金を抑えるコツ

![コスト節約テクニック](images/ch04_07_cost_saving.png){ width=100% }

1. **利用上限を設定してください**（API管理画面で月額上限を設定できます）
2. **無料のローカルLLMを使ってください**（Ollamaなど。性能は落ちますが料金ゼロです）
3. **DeepSeek APIを使ってください**（高性能なのに料金が安いです）
4. **必要な時だけ使ってください**（24時間稼働は料金が跳ね上がります）

上限設定を忘れて1万円飛んだという話もありますので、必ず最初に設定しておいてください。
