# リサーチ結果: AI図解生成 初心者向け入門

## 参考資料の要約

すばる（清水昴）氏のnote記事「AI図解の終着駅：24種の図解パターン×NanobananaProプロンプト集」。
- 24種類の図解パターンをNanobananaPro形式で体系化
- 核心は「バイリンガル・ハイブリッド記述」（構造は英語、テキストは日本語）
- Subject / Layout / Visuals / Style の4要素でプロンプトを構成
- 元薬剤師 → AI出版プロデューサーとして数千回のプロンプト検証を経て開発

## Layer 3: SNS / ショート動画トレンド

### バズキーワード

- 「読むリール」（動画なのに文字を読ませる静止画スライドショー）
- 「フォトモード」（TikTokの複数枚静止画スライドショー）→ 図解コンテンツと高い親和性
- 「AI図解」「インフォグラフィック」
- 「NanobananaPro」「Gemini画像生成」

### トレンドの切り口

- Instagram「読むリール」が定着 → 図解コンテンツとの相性抜群
- TikTok「フォトモード」で図解投稿が急増 → 動画制作リソースなしで参入可能
- Z世代・α世代は情報を「数秒で判断」→ 図解の需要が急上昇
- 2026年はAI×SNSがさらに密接に → AIで図解を量産する時代
- リール広告はフィード広告と比較してエンゲージメント率が約2.5倍

## Layer 5: 読者の悩み・ニーズ

### よくある悩み

1. 日本語の文字が崩れる・潰れる（最も多い悩み）
2. レイアウトが意図通りにならない
3. 色やデザインがイメージと違う
4. どの図解パターンを使えばいいかわからない
5. プロンプトの書き方がわからない
6. 毎回プロンプトを一から書くのが面倒
7. 生成した図解の品質にバラつきがある
8. 英語でプロンプトを書かないとダメ？

### 初心者がつまづく壁

- 文字の詰め込みすぎによる崩れ
- 行間・余白・コントラストの不足
- 「短く具体的」なプロンプトが書けない
- 英語の構造指示と日本語テキストの使い分けがわからない
- 図解パターンの選び方がわからない
- 後編集（Canva等）との連携方法がわからない

### 対処法として知られている技術

- ゴシック体・太めフォントを指定する
- 余白・縁取り・コントラストをプロンプトに明示
- 英語で配置を確定してから日本語に差し替える
- 背景とテキストを分離して生成する
- 段階的に情報を足す（一気に長文にしない）

## 総合分析: 本書の方向性

### 推奨する切り口

「AI図解の作り方」に特化した初心者向け実践書。NanobananaPro形式の24種パターンをコピペで使えるプロンプト集として提供しつつ、「なぜこう書くと上手くいくのか」を初心者にもわかりやすく解説する。

### 盛り込むべきポイント

1. AIが図解を苦手とする理由（空間認識と言語の分離）→ 初心者の「なぜ？」に答える
2. バイリンガル・ハイブリッド記述の原理（英語で構造、日本語でテキスト）
3. 24種パターンの「いつ・どれを使うか」判断フローチャート
4. 文字崩れを防ぐ具体的テクニック（フォント指定、余白、コントラスト）
5. コピペで使えるプロンプトテンプレート
6. SNS（Instagram・TikTok）での活用法 → 「読むリール」「フォトモード」
7. Canva等での後編集テクニック

### 避けるべきこと

- 専門用語の羅列（初心者向けなので平易な表現で）
- AI技術の深い解説（原理より「使い方」にフォーカス）
- 競合と同じ「AI技術の概説書」にならないこと
- プロンプト例の省略（すべてフルテキストで提供）
