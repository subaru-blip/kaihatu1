## 3. バズるタイトル 15の黄金パターン

リサーチから抽出した、漫画動画で特に効果が高いタイトルパターン。

### パターン1: 「もしも○○だったら／したら」型
**心理効果**: 好奇心ギャップ（答えが知りたくなる）

```
もしも税金がなかったら
この世界に女性が1割しかいないと、どうなるのか？
男性だけの刑務所に女が入れられたら
生き埋めになったらどうすればいいのか？
```

**ポイント**: 日常にありえない状況を提示→「結果が気になる」でクリック

---

### パターン2: 「○○の末路」型
**心理効果**: 因果応報への期待、結末への好奇心

```
詐欺で生きる男の末路
念願のハイスペック旦那を捕まえ有頂天になっていた女の末路…
○○し続けた人間の末路
```

**ポイント**: 「末路」は破滅を暗示→悪者がどうなるか見たい心理

---

### パターン3: 「○○を漫画にしてみた」型
**心理効果**: 実話感+視覚化の期待

```
金正恩はどんな生活をしているのか漫画にしてみた
前世の記憶を持つ3歳の少年が、自分を殺した犯人を捕まえた実話を漫画にしてみた
TVロケで本当に起こってしまった心霊現象を漫画にしてみた
死刑囚の最後の1時間をマンガにした
```

**ポイント**: 「実話」「本当に」で信憑性UP→普段見れない世界を可視化

---

### パターン4: 「○○にしかわからない○○」型
**心理効果**: ターゲット共感+排他的帰属意識

```
ブスにしかわからない感情5選
男子にしかわからない感覚
社会人にしかわからないこと
```

**ポイント**: 「自分のこと言ってる！」と思わせる→共感クリック

---

### パターン5: 「○○した結果…」型（結末を隠す）

```
年収差がある同期の男が自信家だったが、翌日から変化が…

```

**ポイント**: 「…」で途中で切る→結末が気になってクリック

---

### パターン6: 「セリフ埋め込み」型（会話形式）
**心理効果**: 臨場感、ドラマ性

```
嫁両親「娘が流産とは」俺「1年以上行為していない」

```

**ポイント**: 会話のやり取りで一瞬でストーリーを伝える→衝撃展開が見える

## 4. タイトル設計の基本ルール

### 文字数
- **最適文字数**: 25〜30文字（28文字超はスマホで省略される）
- **再生回数トップ1000動画の平均**: 約47文字（ただし重要情報は前半40文字以内に）
- **漫画動画では**: 短めのインパクト重視 or 状況説明型の長めタイトル、両方が有効

### キーワード配置
- **最重要キーワードは文頭に配置**（検索アルゴリズムが重視）
- キーワードは3つまでに絞る
- YouTube検索サジェストからキーワード候補を調査

### 記号の効果的な使い方
| 記号 | 用途 | 例 |
|---|---|---|
| 【】 | キーワード強調・ジャンル表示 | 【漫画】【実話】【衝撃】 |
| …（三点リーダー） | 結末を隠す・余韻を残す | 翌日から変化が… |
| ！？ | 驚き・疑問の強調 | え？？ |
| 「」 | セリフの埋め込み | 友人「お前浮気はないだろ」 |
| → | 因果関係の示唆 | パワハラ上司→即クビ |

### CTR（クリック率）の目安
- 一般的な平均値: 4〜5%
- 良好: 7〜8%
- 優秀: 10%以上

## 5. パワーワード辞典

タイトルに入れるとクリック率が上がるワード集。心理学的には「限定」「秘密」「衝撃」などの言葉に対して脳内でドーパミンが分泌され、強い関心が喚起されることが実証されている。

**使い方ルール**: 1つのタイトルに**1〜2個**が適切。詰め込みすぎは逆効果。毎回「緊急」「重大発表」で始めるとインフレを起こして効果が薄れる。

### 感情刺激・衝撃系
`驚愕` `衝撃` `唖然` `炎上覚悟` `ハプニング` `鳥肌` `号泣` `激怒` `絶句` `感動` `感動的` `癒される` `涙` `神回` `炎上`

### 好奇心・秘密系
`裏話` `有料級` `秘密` `ここだけの話` `神業` `知られざる` `真実` `の闇` `裏側` `裏技` `なぜ` `理由`

### 緊急・危機感系
`ダメ絶対` `やらないと損` `知らないと怖い` `警告` `今すぐ` `限定` `最後の` `危険` `知らなきゃ損` `やってはいけない`

### 信頼・実証系

## 7. 心理学テクニック一覧

タイトルに使える心理学的手法と、その活用方法。

| 心理テクニック | 説明 | タイトル活用例 |
|---|---|---|
| **好奇心ギャップ** | 「知っていること」と「知りたいこと」の間の空白を作る | 「○○した結果…」（結末を隠す） |
| **カリギュラ効果** | 禁止されると逆にやりたくなる | 「ガキは見るな」「検索してはいけない」 |
| **損失回避** | 損を避けたい心理 | 「知らないと損する」「やらないと後悔する」 |
| **バーナム効果** | 自分に当てはまると感じる | 「○○な人の特徴」「○○にしかわからない」 |
| **バンドワゴン効果** | みんながやっているから自分も | 「プロがやっている」「実際にバズった」 |
| **ネガティブバイアス** | ポジティブよりネガティブ情報に反応 | 「の末路」「の闇」「衝撃の真実」 |
| **ツァイガルニク効果** | 未完了のものが気になる | 「…」で途切れるタイトル |
| **希少性の法則** | 珍しいものに価値を感じる | 「初公開」「ここだけ」「限定」 |

| **ギャップ効果** | 予想とのズレが印象に残る | 「ブラック企業のテンションで優しい会社」 |

## 10. 重要な注意点

### やってはいけないこと
- **内容とタイトルの不一致**: 視聴維持率が下がり、アルゴリズムに不利
- **過激すぎる煽り**: 「世界一」「絶対」「確実」は信頼を落とす
- **キーワード詰め込み**: スパム判定のリスク
- **長すぎるタイトル**: スマホで見切れて意味不明になる

### 漫画動画特有の傾向
- **1話完結**がメイン→毎回タイトルで新規視聴者を獲得する必要がある
- **サムネとタイトルのセット**で考える（サムネ＝ビジュアルの衝撃、タイトル＝文字の補足）
- **スカッと系・もしも系・あるある系**の3ジャンルが安定して高再生
- **ネガティブ系タイトル**の方がポジティブ系より平均CTRが高い
- **セリフ埋め込み型**は漫画動画との相性が特に良い（漫画＝吹き出し文化）

## 11. タイトル生成フロー

```
Step 1: ジャンルを決める
  └ 闇社会 / スカッと / もしも / エモい / コメディ / 恋愛 / 雑学

  └ 最低1つ、できれば2つ

Step 4: 心理テクニックを確認
  └ 好奇心ギャップ or カリギュラ効果 が最も汎用性高い

Step 5: 文字数チェック（28文字以内推奨）

Step 6: チェックリストで最終確認
```

## 13. サムネイルとタイトルの関係

- サムネイルとタイトルは**補完関係**にすべき（同じ情報の繰り返しではなく、サムネイルで伝えきれない情報をタイトルで補う）
- サムネイル＝ビジュアルの衝撃、タイトル＝文字情報の補足
- サムネイル内の文字は**20文字以内**に収めると複数のサムネイルが表示された際にも瞬時に読み取りやすい
- A/Bテストを繰り返しながら勝ちパターンを見つけることが重要
