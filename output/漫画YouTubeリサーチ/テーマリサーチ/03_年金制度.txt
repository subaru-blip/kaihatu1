================================================================

1. 年金制度の基本構造
----------------------------------------------------------------

【国民年金・厚生年金の仕組み】

■国民年金（基礎年金）= 1階部分
- 日本に住む20歳以上60歳未満のすべての人が強制加入（国民皆年金）
- 2025年度の保険料：月額17,510円（毎年度改定）
- 満額受給：月額69,308円（2025年度）
- 受給資格：保険料納付済期間等が10年以上

■厚生年金 = 2階部分
- 会社員・公務員が加入（第2号被保険者）
- 保険料率は18.3%（労使折半なので本人負担は9.15%）で2017年に固定
- 報酬に比例して年金額が決まる（平均標準報酬月額 x 加入期間で計算）
- 国民年金に上乗せして支給される

【3階建て構造の解説】

| 階層 | 名称 | 対象者 | 特徴 |
|------|------|--------|------|
| 1階 | 国民年金（基礎年金） | 全国民（20-60歳） | 定額給付 |
| 2階 | 厚生年金 | 会社員・公務員 | 報酬比例 |
| 3階 | 企業年金・iDeCo等 | 任意加入 | 私的年金 |

3階部分には、確定給付企業年金（DB）、企業型確定拠出年金（DC）、個人型確定拠出年金（iDeCo）、国民年金基金などがある。

【支払い額と受給額の基本】
- 国民年金の平均受給額：月額約57,700円（令和5年度実績。満額ではない人が多いため満額69,308円より低い）
- 厚生年金の平均受給額：月額約145,638円（国民年金分を含む）
- モデル世帯（夫：会社員40年、妻：専業主婦）：月額232,784円（2025年度）


2. 知らないと損する事実
----------------------------------------------------------------

【繰り上げ受給と繰り下げ受給の損得】

■繰り上げ受給（60歳~64歳で開始）
- 1か月早めるごとに0.4%減額（1962年4月2日以降生まれ）
- 60歳から受給すると24%減額（一生涯続く）
- 損益分岐点：80歳10か月。これ以上生きると65歳受給開始の方が得
- 男性の66.2%、女性の83.6%がこの年齢まで生存する

■繰り下げ受給（66歳~75歳で開始）
- 1か月遅らせるごとに0.7%増額
- 70歳開始で42%増、75歳開始で最大84%増
- 損益分岐点（70歳開始の場合）：81歳11か月
- 損益分岐点（75歳開始の場合）：86歳11か月

■具体例：月額15万円の人が70歳繰り下げ
- 65歳開始：月15万円
- 70歳開始：月21.3万円（42%増）
- ただし税金・社会保険料が増えるため、手取りベースの増額率は額面ほどではない

【付加年金（月400円で大幅増額）】
- 対象者：国民年金第1号被保険者（自営業・フリーランス等）、任意加入被保険者
- 保険料：月額たったの400円
- 受給額：200円 x 納付月数（年額）
- 例：30年間（360か月）納付した場合
  - 総支払額：400円 x 360か月 = 144,000円
  - 年間増額：200円 x 360か月 = 72,000円
  - たった2年で元が取れる（その後は純粋なプラス）
- 注意：国民年金基金との併用は不可

【免除制度の活用法】

| 免除区分 | 保険料 | 年金への反映 |
|----------|--------|-------------|
| 全額免除 | 0円 | 1/2が反映 |
| 3/4免除 | 約4,380円 | 5/8が反映 |
| 半額免除 | 約8,760円 | 6/8が反映 |
| 1/4免除 | 約13,130円 | 7/8が反映 |
| 納付猶予 | 0円 | 反映なし（資格期間には算入）|

- 未納のまま放置するのが最悪の選択。免除申請すれば受給資格期間に算入される
- 10年以内なら追納可能（追納すれば満額に近づく）
- 免除期間中も障害年金・遺族年金の受給権は守られる

【遺族年金・障害年金の存在】

■遺族年金
- 加入者が亡くなった際、遺された家族に支給される
- 遺族基礎年金（子のある配偶者または子）+ 遺族厚生年金
- 申請しないと支給されない（知らないともらえない）

■障害年金
- 病気やけがで障害状態になった場合に支給
- 障害基礎年金（1級：年約100万円、2級：年約80万円）
- 障害厚生年金はさらに上乗せ
- 65歳までに請求する必要がある
- 非課税（所得税・住民税がかからない）

【年金の税金問題】
- 老齢年金は「雑所得」として課税対象
- 65歳未満：年108万円以上で課税、65歳以上：年158万円以上で課税
- 公的年金等控除により一定額までは非課税
- 繰り下げで年金額が増えると、税金・社会保険料も増加し手取りの増額率は額面より低い
- 障害年金・遺族年金は非課税
- 年金収入400万円以下かつ他の所得20万円以下なら確定申告不要

【未納率・免除率の実態】

- 全額免除・猶予者：約592万人
- 20~24歳の納付率が88%と最も高い（学生納付特例を除く計算方法の影響）
- 25~29歳で77%に低下

- 2024年度財政検証：所得代替率61.2%
- 厚生年金積立金：約5年分の支給額を保有
- 国民年金積立金：約4年分の支給額を保有
- GPIF累積収益額：約153.6兆円（2001~2024年度）
- GPIF運用資産：約249.8兆円
- 直近5年間の累計運用収益：約98兆円


4. よくある勘違い
----------------------------------------------------------------

【「年金は破綻する」は本当か？】
結論：制度としての破綻はしない。ただし給付水準は下がる。

理由：
1. マクロ経済スライドが導入済み。年金支給額の伸びを自動的に抑制し、財政を安定させる仕組み。2025年度までに6回発動。
2. 保険料上限が固定（厚生年金18.3%）されており、その範囲で給付を調整する仕組み。
3. 国庫負担（税金）が基礎年金の1/2を支えている。
4. GPIFの巨額の積立金（約250兆円）がバッファ。
5. 「破綻」ではなく「給付水準の低下」が正確な表現。基礎年金の調整は2052年度まで続く見込み。

【「払わない方が得」の嘘】
払わないデメリット：
- 障害年金・遺族年金の受給権を失う（これが最大のリスク）
- 将来の老齢年金がゼロまたは大幅減額
- 年金は終身給付なので長生きリスクに対応できる唯一の仕組み
- 全額免除でも1/2は年金に反映される。「未納」と「免除」は全く別物

【「自分で運用した方がいい」の落とし穴】
- 年金は「保険」であり「投資商品」ではない
- 自分で運用する場合に年金が持つ以下の機能をカバーできない：
  - 終身給付（何歳まで生きても受給し続ける）
  - インフレ対応（物価スライド）
  - 障害時の保障
  - 死亡時の遺族保障
  - 国庫負担（税金による補填）分は自前では得られない
- 投資は元本割れリスクがあるが、年金には一定の保障がある

【世代間格差の実態】
- 1940年生まれ：約3,460万円の「得」
- 2010年生まれ：約2,830万円の「損」
- 差額は約6,290万円
- ただし、この計算は「保険」としての価値（障害年金・遺族年金・長生きリスクカバー等）を無視した金銭的な損得計算のみ
- 若い世代でも保険料の2倍以上は受け取れる試算がある


5. 面白い・意外な事実
----------------------------------------------------------------

【年金制度の歴史（ドイツ・ビスマルクが起源）】
- 1889年：ドイツ帝国初代首相ビスマルクが世界初の強制加入年金制度を創設
- 背景：社会主義運動の拡大に対する対抗策。弾圧（ムチ）と社会保障（アメ）の両面戦略
- 1880年代にドイツで事故保険（1884年）、疾病保険（1883年）、年金保険（1889年）が相次いで誕生

【日本の年金制度開始時の経緯】
- 1941年：労働者年金保険法が公布（ナチス・ドイツの年金制度を参考）
- 当初は10人以上の事業所の男子筋肉労働者のみが対象
- 1944年：「厚生年金保険」に改称、対象を女性・事務職にも拡大
- 1961年：国民年金制度の開始により「国民皆年金」達成
- 農業従事者や自営業者はそれまで年金制度の対象外だった
- 戦時中の導入には「戦費調達」の側面もあったとされる

【積立方式と賦課方式の違い】

| | 賦課方式 | 積立方式 |
|--|---------|---------|
| 仕組み | 現役世代の保険料で高齢者の年金を支払う | 自分の将来の年金を自分で積み立てる |
| インフレ耐性 | 強い（その時の賃金に連動） | 弱い（積立金の価値が目減り） |
| 少子高齢化の影響 | 大きい | 小さい |
| 日本の制度 | 基本は賦課方式 + 積立金の活用 | 企業年金等は積立方式 |

【GPIFの運用実績】
- 世界最大級の機関投資家（運用資産約250兆円）
- 2001年度からの累積収益：約153.6兆円
- 直近5年間で約98兆円の収益
- 2025年度第3四半期（10~12月）の収益率：+5.84%
- ポートフォリオ：国内株式25%、外国株式25%、国内債券25%、外国債券25%（基本配分）
- 長期分散投資の好例として世界的に注目されている


6. 漫画ストーリー案
----------------------------------------------------------------

【パターンA:「年金なんて払わない」と言っていた若者の末路】

登場人物：タクヤ（25歳、フリーランスのWebデザイナー）

■序盤：
- タクヤ「年金なんてどうせ破綻するし、払うだけ無駄。自分で投資した方がよっぽどマシ」
- 国民年金の納付書をゴミ箱に捨てる
- SNSで「年金は情弱が払うもの」と投稿、仲間内でも未納を自慢

■中盤（転機）：
- 30歳で交通事故に遭い、片足に重度の障害が残る
- 障害年金を申請しようとするが「未納期間が長すぎて受給資格なし」と告げられる
- 障害基礎年金2級なら年約80万円、一生もらえたはずなのに...
- 一方、同世代の友人ユウキは免除申請をしていたため障害年金の受給資格あり

■終盤：
- タクヤが未納でなく「免除申請」していれば、保険料0円でも受給資格は守れた
- 投資で貯めた500万円は治療費であっという間に消える
- ユウキ「年金は投資じゃなくて保険なんだよ。障害年金も遺族年金もある。知らないのが一番の損だ」

■ラスト：「年金は"未来の自分への保険"。払えないなら免除申請、これだけは絶対忘れるな」

【パターンB: 知識で得する人と損する人の対比】

登場人物：
- アキラ（60歳、元サラリーマン、年金知識ゼロ）
- サトシ（60歳、元サラリーマン、年金に詳しい）

■序盤：二人とも同じ会社で同じ給料で40年間働き、定年退職

■アキラの選択（知識なし）：
- 「早くもらえるに越したことはない」と60歳で繰り上げ受給
- 月15万円 → 24%減で月11.4万円に（一生この金額）
- 付加年金の存在も知らなかった
- 確定申告不要制度も知らず、毎年無駄に申告

■サトシの選択（知識あり）：
- 70歳まで繰り下げ。それまではパート+貯蓄で生活
- 月15万円 → 42%増で月21.3万円に
- 付加年金も40年納付済み。年間96,000円上乗せ
- 確定申告の控除も適切に活用

■結果：
- 82歳時点の累計受給額（アキラ vs サトシ）を比較
- アキラ：月11.4万円 x 22年 = 約3,010万円
- サトシ：月21.3万円 x 12年 = 約3,067万円（+付加年金約115万円）
- 83歳以降はサトシが圧倒的に有利、差は開く一方

【ラストの教訓・具体的アクション】
1. ねんきん定期便を毎年必ず確認する
2. 免除制度は未納のまま放置より100倍マシ
3. 付加年金（月400円）はフリーランス最強のオプション
4. 繰り下げ受給は健康状態と資産を見て判断
5. 障害年金・遺族年金の存在を家族にも伝えておく
6. 「ねんきんネット」に登録して自分の見込み額を把握する
